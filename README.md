# 💌 Formulário de Mensagem

Este é um pequeno projeto que desenvolvi para praticar **HTML** e **CSS**.  
A ideia foi criar um formulário de contato bonito, limpo e fácil de usar — algo que poderia fazer parte de um site real.

---

## 🖥️ Sobre o projeto

O formulário permite que o usuário envie uma mensagem preenchendo:
- Nome  
- E-mail  
- Assunto  
- Texto da mensagem  

Tudo isso com um design moderno, fundo em **gradiente roxo** e um botão de envio que muda de cor quando o mouse passa por cima.

---

## 🧩 Tecnologias usadas

- **HTML5** para a estrutura da página  
- **CSS3** para o estilo e o layout
